~Change your WiFi password~Wi-Fi Hacking

This project was made to showcase the major security flaws in wireless networks. Most ISPs configure default passwords that make it easy for consumers to access their network, this consumer-based configuration creates a vulnerability that malicious actors can exploit. This project is a tool that targets Charter Spectrum networks by exploiting the default password structure. This is a serious security flaw that can lead to stolen personal information, legal issues, and a slower bandwidth connection.

Take a look at this project and learn how this security flaw was exploited and how you can protect yourself.

Programming with Python:

All the tools I used are run through the command line interface(CLI) also known as the terminal. So I used the SubProcess Python Library to run these tools in the program as if they were being manually typed and processed by a person. The SubProcess Library has useful features that can breakdown a terminal command into different manageable pieces - input arguments, output, error codes, & return codes. Most importantly this library can be used to capture the output of commands run through the terminal, which can be parsed to extract the significant information.

Examples of code:
subprocess.run("command", capture_output=True, text=True) subprocess.Popen("command", stdout:subprocess.PIPE,)



Breaking the Password Structure

Understanding the Password Structure:

The Charter Spectrum password structure:

  • Adjective + Noun + 3 Numbers
  • All lowercase
  • Examples:
    • heavyapple547
    • mightycheese485

This is so the password is easy to remember, easy to enter, and easy to communicate.

Gathering words:

Using a simple Google search, I was able to find lists of adjectives and nouns that I was able to put together in a program that'll generate password combinations.

Generating the wordlist:

The combination wordlist was generated by a separate Python program that organizes the collected words into arrays then uses double for-loops to run through every possible combination of adjectives and nouns.

How To Protect Youself:

Simple steps to protect yourself:
  1. Change your password!
    • Use a readable password
    • Include a few symbols and numbers
    • Easy to remember, easy to enter, easy to communicate
  2. Change your network name
    • Change your Wi-Fi network name to something unique
    • Makes it hard to identify the ISP and any possible passwords
    • Be creative, have fun
  3. Hide your network
    • Disable the network broadcast – this means the Wi-Fi details must be entered manually
    • Most effective way of preventing outside threats to your network
